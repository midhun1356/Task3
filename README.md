# E-Commerce App 🛒

A full-stack E-Commerce web application developed using HTML, CSS, JavaScript, Node.js, Express.js, and MySQL.

The application allows users to register, login, view products, and add products to their cart.

---

## 🚀 Features

- User Registration
- User Login Authentication
- View Products
- Add Products to Cart
- Cart Management using Local Storage
- Backend API using Node.js and Express.js
- MySQL Database Integration
- Responsive User Interface

---

## 🛠️ Technologies Used

### Frontend
- HTML5
- CSS3
- JavaScript

### Backend
- Node.js
- Express.js

### Database
- MySQL
